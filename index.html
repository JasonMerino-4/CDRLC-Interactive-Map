How to Use CDLRC Map

Search & Filter
Use the searchbar to show a specific room. Or use the filters to find a type of room
Browsing Floors
Use the floor buttons to switch between floors. The selected floor will be used when filters are applied.
Get Directions
Click or input a room as a starting point and another as the destination point. Text direction and visual navigation will activate.
